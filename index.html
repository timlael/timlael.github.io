Welcome to timlael.github.io

 • Posted by Tim M. Lael on May 18, 2017  • 

 This is a collection of some of my academic work created throughout the course of my studies at The University of Missouri - St. Louis which concluded May 13, 2017 when I was awarded a Bachelor of Science - Computer Science.

 Most of the programs (which can be viewed by visiting the Projects tab above) are written in C unless requirements dictated another language be used. Early on, I chose C as my primary language because of the challenge of implementing project requirements which would have been simpler using an object-oriented language. After all, I was there to learn, so why not make it at least a little challenging. Later, I was pleasantly surprised when the utility of C and its close relationship with *nix proved very useful in Operating Systems classes when implementing memory management, process scheduling, I/O and other low level tasks.

 Unfortunately, my professional work is not mine to publish. It consists of mainly SQL scripts with a fair amount of HTML, JavaScript, C, Shell Scripts and many Oracle forms and database maintenance related scripts.
